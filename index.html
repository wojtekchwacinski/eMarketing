O projekcie

Ten layout pokazuje, jak może wyglądać prosty dashboard: kliknięcia, liczba sesji, poziom aktywności i historia zdarzeń. W przyszłości przycisk „Połącz urządzenie” można przepiąć pod realną integrację.

Przyszłe rozszerzenia:

API • WebSocket • Bluetooth/Wi-Fi • logowanie • eksport danych
